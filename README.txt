Como ejecutar la pagina web por mi cuenta.

1.- Descargar los archivos de la pagina web (carpetas css, js, img y archivos html y php)

2.- descargar e instalar xampp y php

3.- abrir tu disco local C: (o donde instalaras xxampp) y buscar la carpeta llamada xampp y entrar a ella

4.- en la carpeta xampp entrar a la carpeta htdocs

5.- en la carpeta htdocs, crear una carpeta llamada: PIA_1A, y en esta carpeta nueva colocar los archivos descargados de la pagina web (los archivos php y html, las carpetas css, js e img)

6.-Despues abrir la aplicacion xampp, dar Start a las opciones que dicen apache y MySql

7.- alado del boton de start de MySql dar click en el boton que dice admin esto abrira en tu navegador phpmyadmin, para crear la base de datos.

8.- dar click en el icono de la circunferencia, donde dice nuevo, para empezar a crear la base de datos.

9.- en nombre de base de datos pones: betterwere y le das a crear, despues en la parte superior de la base de datos veras un boton que dice:  SQL, le das click.

10.- en esta seccion poner el query exportado de nuestra base de datos y le das a continuar, esto creara todas las tablas y registros de nuestra base de datos.

11.- finalmente en cualquier pestaña de tu navegador pon localhost/PIA_1A/index.php

12.- Ya podras ver la pagina completamente funcional
